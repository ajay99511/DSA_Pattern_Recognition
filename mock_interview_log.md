# 📊 Mock Interview Log

> Track every mock session with quantitative metrics. No vague "it went okay" — measure everything.

---

## Session Template

Copy this for each mock session:

```
## Mock Session #___ — [Date]

### Setup
- **Problem**: [Name] (LC #XXX) — [Easy/Medium/Hard]
- **Timer Started**: [HH:MM]
- **Pattern Expected**: [What you think before starting]

### Execution Metrics
- **Time to Pattern ID**: _____ seconds (target: <60s)
- **Time to Working Solution**: _____ minutes (target: <25 min medium, <40 hard)
- **Pattern Identified**: [correct / wrong → what you picked vs. actual]
- **Solution Accepted**: [yes / no / partial]
- **Bugs Found During Testing**: [list each bug]

### Communication Score (self-rated 1-5)
- [ ] Did I restate the problem? ___/5
- [ ] Did I discuss examples/edge cases first? ___/5
- [ ] Did I explain approach BEFORE coding? ___/5
- [ ] Did I narrate while coding? ___/5
- [ ] Did I analyze complexity at the end? ___/5
- **Average**: ___/5

### Failure Recovery
- **Was I stuck?**: [yes/no]
- **Which mode?**: [Blank / Stuck / TLE / None]
- **Did the recovery protocol help?**: [yes/no/partially]
- **Did I pivot?**: [yes/no — from what to what?]

### Key Takeaway
> [One sentence: what's the #1 thing to remember from this session]

---
```
